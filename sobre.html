Sobre

Este site foi desenvolvido como parte de um projeto acadêmico para a faculdade, com o objetivo de promover e organizar o trote solidário. O trote solidário é uma iniciativa que incentiva a participação dos estudantes em atividades filantrópicas e comunitárias, ao invés de práticas tradicionais de trote.

Os estudantes envolvidos no projeto trabalham juntos para criar soluções que possam contribuir para a sociedade, promovendo ações como doações de alimentos, sangue e muito mais. O objetivo principal é impactar positivamente as comunidades locais.

Este projeto representa a colaboração de estudantes do curso de CC e SI, que trabalharam na criação de um site funcional e informativo para facilitar a organização e divulgação das atividades do trote solidário.
